## 📋 Module Summaries

### Module 1 — Digital Literacy Awareness (Task 1)
Created a one-page infographic using **Canva** covering: what digital literacy is, essential tools for B.Tech students, safe internet practices, professional online presence, and email etiquette. The goal was to design a resource that batchmates could easily understand and act on.

### Module 2 — Building a Digital Portfolio (Task 2)
Set up a professional presence on **GitHub** — including a profile README, project repositories, and this digital literacy project repository. GitHub serves as a living portfolio that showcases practical skills to future employers and internship coordinators.

### Module 3 — Coding & Collaboration Platforms (Task 3)
Explored **HackerRank** by completing beginner coding challenges, and created a 5-question **Digital Literacy Awareness Quiz** using Google Forms. The quiz response data is linked to a Google Sheet for tracking.

### Module 4 — Email Etiquette & Communication (Task 4)
Drafted two professional emails — one requesting an assignment deadline extension and one expressing interest in a summer internship. Also created a Social Media Do's and Don'ts checklist for responsible online behaviour as a college student.

### Module 5 — Cybercrime Awareness (Task 5)
Researched **UPI / Online Payment Fraud**, wrote a detailed case study with a step-by-step breakdown of how such scams work, and created a prevention checklist with 8+ actionable safety tips — including two UPI-specific tips and the national reporting channel.
